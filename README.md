# HexagramInsight

A comprehensive repository that provides information on the 64 hexagrams of the I Ching, including their names, upper and lower trigrams, and detailed explanations of each hexagram's meaning and significance

一個全面的資料庫，提供《易經》六十四卦的相關資訊，包括卦名、上下卦以及每個卦象的詳細解釋和意義。

## 計算方式

https://clonn.github.io/yi-insight/caculator.html

1. 誠心默念你的問題
2. 將心目中浮現的第一印象，給予三個欄位，連續三個數字
3. e.g. 111 / 222 / 333
4. 按下計算結果
5. 如果不知道怎麼解盤，歡迎截圖發 issue 給我。
